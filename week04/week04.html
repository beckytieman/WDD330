Week 04 Notes

Reading Ch8: Forms

  • built in form validation can be limited; get comfortable with custom script
  • be careful with reset buttons - can be considered bad for usability
  • JavaScript form validation shorts

Reading Ch12: Object Oriented Programming

  • Encapsulation: details are kept from user, program parts are inside 'objects'
  • Polymorphism: reusable parts; the same things is usable for different objects
  • Inheritance: features(properties and methods) from one object can be inherited to another. You can also add additional features to the next object

Reading Ch15: Modern Javascript

    Modular JavaScript

  • self-contained code with functions/methods that can be used elsewhere (think libraries)
  • keeps code organized in seperate, reusable files.
  • modules has it's own global scope
