Week 02 Notes

Reading Ch2: Programming Basics

  • Using variables with non-primitive data types - const will not be immutable (arrays, object, etc..)
  • Quit using var all the time - both let and const have block scope(their value only exists inside the block they are declared in)
  • Template Literals - pros: use both types of quotes, JS expressions can be inserted into string

Reading Ch3: Arrays, Logic, and Loops

  • Destructuring Arrays gets rid of the temp variable!
  • for-of loops iterate over an array 'cleaner' than for loop (also use for sets/maps)

Reading Ch4: Functions

  • Keep your code DRY(don't repeat yourself)
  • Arrow Functions: =>
    • pros: good for short, one line functions (how big is too big for using arrow functions?)
    • parameters come BEFORE the arrow and the main function body comes AFTER
    • Always anonymous - assign to a variable if you want to refer to them
    • multiple parameters must go in ( )
    • if NO parameters, empty () goes before arrow
  • Callbacks are just functions passed as an argument to another function
